== nowrap

This option makes the editable area behave like a `<pre>` tag, and adds a horizontal scroll instead of wrapping text.

*Type:* `Boolean`

*Default Value:* `false`

*Possible Values:* `true`, `false`

===== Example

`js
tinymce.init({
  selector: 'textarea',  // change this value according to your HTML
	nowrap : true
});
`

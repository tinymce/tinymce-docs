### `spellchecker_language`

This option specifies the default language used by Spell Checker Pro.

**Type:** `String`

**Default Value:** `'en_US'`

#### Example: Using `spellchecker_language`

```js
tinymce.init({
  selector: 'textarea',
  plugins: 'tinymcespellchecker',
  spellchecker_language: 'de'
});
```

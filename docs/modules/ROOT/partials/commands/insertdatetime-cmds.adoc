|===
| Command | Description

| mceInsertDate
| Inserts the current date as a human readable string.

| mceInsertTime
| Inserts the current time as a human readable string.
|===

*Examples*

[source,js]
----
tinymce.activeEditor.execCommand('mceInsertDate');
tinymce.activeEditor.execCommand('mceInsertTime');
----

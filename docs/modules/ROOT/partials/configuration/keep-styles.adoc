[[keep_styles]]
== keep_styles

The *keep_styles* option will keep the editor's current text style when a user presses `enter/return`.

*Type:* `Boolean`

*Default Value:* `true`

*Possible Values:* `true`, `false`

[discrete#example]
===== Example

[source,js]
----
tinymce.init({
  selector: 'textarea',  // change this value according to your HTML
  keep_styles: false
});
----

== Accompanying Premium plugin changes

The following premium plugin updates were released alongside {productname} 6.7.2.

[[powerpaste-622]]
=== PowerPaste 6.2.2

The {productname} 6.7.2 release includes an accompanying release of the **PowerPaste** premium plugin.

**PowerPaste** 6.2.2 includes the following fix.

==== Stopped pasting comments from Microsoft Word documents when Safari is the host browser

**PowerPaste** 6.2.1 included xref:6.7-release-notes.adoc#powerpaste-621[a fix] that stopped comments associated with a copied portion of a Microsoft Word document being pasted into a {productname} document as footnotes. 

This fix had a known issue: it did not work when the host browser for a {productname} instance was Safari.

**PowerPaste** 6.2.2 addresses this known issue.

With this update, when a selection from a Microsoft Word document that includes comments is copied and pasted (⌘-V) into a {productname} instance, the included comments are not set as footnotes even when Safari is the host browser.

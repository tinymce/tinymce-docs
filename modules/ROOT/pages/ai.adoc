The {pluginname} allows a {productname} document to send queries to registered AI APIs and either:

. add the returned results to a document; or
. temporarily store the returned results for later use.

== Basic setup

To add the {pluginname} plugin to the editor, add `{plugincode}` to the `plugins` option in the editor configuration.

For example:

[source,js]
----
tinymce.init({
  selector: 'textarea',  // change this value according to your HTML
  plugins: 'ai',
  toolbar: 'aidialog aishortcuts',
  ai_request: (request) => new Promise(),
  ai_shortcuts: [],
});
----

[IMPORTANT]
.Always send data with an AI API query
----
The {pluginname} should only be invoked when there is a selection in the {productname} editor to send to the registered AI API.

Selecting an active {pluginname} toolbar button, or selecting an active {pluginname} menu item, when there is no selection, or when the {productname} editor is empty, will result in a random — and almost certainly unhelpful — response.
----

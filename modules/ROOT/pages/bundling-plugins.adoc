Example syntax for including the example "<plugincode>" plugin in a bundle using `content.js`for bundling:

[cols='1,1,4']
|===
|Module Syntax |Source |Example

.2+|ES6+
|npm
a|
[source, js]
----
import 'tinymce/plugins/<plugincode>';
----

|`.zip`
a|
[source, js]
----
import '../tinymce/plugins/<plugincode>';
----

.2+|Common JS
|npm
a|
[source, js]
----
require('tinymce/plugins/<plugincode>');
----

|`.zip`
a|
[source, js]
----
require('../tinymce/plugins/<plugincode>');
----
|===

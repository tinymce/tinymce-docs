= Version Compatibility Reference
:description: Premium plugins compatibility with TinyMCE editor versions.
:description_short: Matrix of compatibility between TinyMCE editor versions and premium plugins.
:keywords: tinymce cloud script textarea apiKey

Some of our premium plugins are only compatible with specific versions of TinyMCE.

If you're deploying link:{rootDir}cloud-deployment-guide/editor-and-features.html[both the TinyMCE editor and premium plugins via the Cloud], use the following matrix in conjunction with the optional parameters for specifying editor version and link:{rootDir}cloud-deployment-guide/editor-plugin-version.html[plugin version] to ensure your editor and plugins are compatible.

If you're deploying link:{rootDir}cloud-deployment-guide/features-only.html[premium plugins _only_ via the Cloud], you'll need to ensure you're either only loading plugins supported for your editor version, or that your editor is upgraded to the required release.

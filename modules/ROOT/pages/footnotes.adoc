== Basic setup

[source,js]
----
tinymce.init({
  selector: 'textarea',  // change this value according to your HTML
  plugins: 'footnotes',
  toolbar: 'footnotes footnotesupdate',
});
----

The footnotes section is automatically created and updated whenever an intext footnote is inserted into the editor, and located at the bottom of the editor. An intext footnote can be inserted using the _Insert footnote_ button, located in the _Insert_ menu or on the toolbar.

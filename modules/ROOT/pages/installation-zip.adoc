== Quick start

Download the {productname} .zip package from the link:{download-community}[official download page].

Extract the archive and include the TinyMCE script:

[source,html]
----
<!DOCTYPE html>
<html lang="en">
  <head>
    <script src="/path/to/tinymce/tinymce.min.js"></script>
  </head>
  <body>
    <textarea id="mytextarea"></textarea>
    <script>
      tinymce.init({
        selector: '#mytextarea'
      });
    </script>
  </body>
</html>
----

== Installation steps

1. Download the {productname} .zip package
2. Extract the archive to your project directory
3. Copy the `+tinymce+` folder to your web-accessible location
4. Include the TinyMCE script in your HTML: `+<script src="/path/to/tinymce/tinymce.min.js"></script>+`
5. Initialize {productname} on a textarea or div element

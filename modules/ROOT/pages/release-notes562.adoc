== Accompanying Premium changes

The following premium updates were released alongside {productname} 5.6.2.

=== Enterprise language packs

The {productname} 5.6.2 release includes an accompanying release of the *Enterprise language packs*.

The *Enterprise language packs* include the following bug fix:

* Fixes Swedish translations (`sv_SE`).

For information on the Enterprise language packs, see: xref:content-localization.adoc#language[Localization options - `language`].

== General bug fixes

{productname} 5.6.2 provides fixes for the following bugs:

* Fixed a UI rendering regression when the document body is using `display: flex`.

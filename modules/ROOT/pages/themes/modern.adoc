= Modern theme
:description: Theme that renders iframe or inline modes using the tinymce core UI framework.
:keywords: theme modern
:title_nav: Modern

The `modern` theme is responsible for rendering the editor in iframe or inline mode. This is currently the default theme there is also a lightweight theme called link:{baseurl}/themes/inlite.html[inlite]

[[example]]
===== Example:

[source,js]
----
tinymce.init({
    selector: "textarea.tinymce",
    themes: "modern"
});
----

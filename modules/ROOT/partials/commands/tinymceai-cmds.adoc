[cols="1,3",options="header"]
|===
|Command name |Description

|`+ToggleSidebar+` |Core command. Use `+editor.execCommand('ToggleSidebar', false, 'tinymceai-chat')+` or `+editor.execCommand('ToggleSidebar', false, 'tinymceai-review')+` to toggle the AI Chat or AI Review sidebar.
|===

.Examples
[source,js]
----
// Open the AI Chat sidebar
tinymce.activeEditor.execCommand('ToggleSidebar', false, 'tinymceai-chat');

// Close the current sidebar (pass same ID again to toggle off)
tinymce.activeEditor.execCommand('ToggleSidebar', false, 'tinymceai-chat');

// Open the AI Review sidebar
tinymce.activeEditor.execCommand('ToggleSidebar', false, 'tinymceai-review');
----

[[autocorrect_autocorrect]]
== `+autocorrect_autocorrect+`

This option enables or disables spelling error auto-correction when the {productname} editor is loaded.

When set to `+true+`, spelling error auto-correction is enabled when {productname} is initialized.

include::partial$misc/autocorrect-options-notes.adoc[]

*Type:* `+Boolean+`

*Possible values:* `+true+`, `+false+`

*Default value:* `+true+`

=== Example: using `+autocorrect_autocorrect+`

[source,js]
----
tinymce.init({
  selector: 'textarea',  // change this value according to your HTML
  autocorrect_autocorrect: false
});
----

[[autocorrect_autocorrect]]
== `+autocorrect_autocorrect+`

This option enables or disables spelling error autocorrection and auto-capitalization when the {productname} editor is loaded. When set to `false`, spelling autocorrection and auto-capitalization are disabled when the editor is initialized.

include::partial$misc/autocorrect-options-notes.adoc[]

*Type:* `+Boolean+`

*Default value:* `+true+`

*Possible values:* `+true+`, `+false+`

=== Example: Using `+autocorrect_autocorrect+`

[source,js]
----
tinymce.init({
  selector: 'textarea',  // change this value according to your HTML
  autocorrect_autocorrect: false
});
----

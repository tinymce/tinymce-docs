[[autocorrect_capitalize]]

== `+autocorrect_capitalize+`

This option allows you to specify the 

*Type:* `+Boolean+`

*Default value:* `+false+`

=== Example: Using `+autocorrect_capitalize+`

[source,js]
----
tinymce.init({
  selector: 'textarea',  // change this value according to your HTML
  autocorrect_capitalize: true
});
----

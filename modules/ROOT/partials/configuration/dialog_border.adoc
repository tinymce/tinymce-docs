[[border]]
=== `border`

include::partial$misc/admon-requires-6.6v.adoc[]

`+border+` is an optional property, available to add to the `iframe` dialog component.

The property has a default value of `+false+`.

When this property is set to true, a border displays around the `iframe` component.

This componenet is also highlighted when it takes focus (by, for example, using the Tab key to move focus to iframe).

*Type:* `+String+`

*Possible values:* `+true+`, `+false+`

*Default value:* `+false+`

=== Example: using `border`

// This should be a working configuration. Please test.
[source,js]
----
tinymce.init({
  selector: 'textarea',  // change this value according to your html
  border: 'bottom'
});
----

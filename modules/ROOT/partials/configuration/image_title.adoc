[[image_title]]
== `+image_title+`

This options allows you enable the image title input field in the image dialog.

*Type:* `+Boolean+`

*Possible values:* `+true+`, `+false+`

*Default value:* `+false+`

=== Example: using `+image_title+`

[source,js]
----
tinymce.init({
  selector: 'textarea',  // change this value according to your HTML
  plugins: 'image',
  toolbar: 'image',
  image_title: true
});
----

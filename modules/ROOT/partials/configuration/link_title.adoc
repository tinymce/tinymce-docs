[[link_title]]
== `+link_title+`

This options allows you disable the link `+title+` input field in the `+link+` dialog.

*Type:* `+Boolean+`

*Possible values:* `+true+`, `+false+`

*Default value:* `+true+`

=== Example: using `+link_title+`

[source,js]
----
tinymce.init({
  selector: 'textarea',  // change this value according to your HTML
  plugins: 'link',
  toolbar: 'link',
  link_title: false
});
----

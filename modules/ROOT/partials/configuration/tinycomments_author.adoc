[[tinycomments_author]]
== `+tinycomments_author+`

This option sets the author id to be used when creating or replying to comments.

Type : `+String+`

Default Value : `+'Anon'+`

=== Example: Using `+tinycomments_author+`

[source,js]
----
tinymce.init({
  selector: '#textarea',
  tinycomments_author: 'embedded_journalist',
  tinycomments_mode: 'embedded'
})
----

[[toc_class]]
=== `toc_class`

With `toc_class` you can change the class name that gets assigned to the wrapper `div`. Please note that you will have to alter xref:boilerplate-content-css.adoc[Boilerplate Content CSS] accordingly.

*Type:* `String`

*Default Value:* `mce-toc`

==== Example: Using `toc_class`

[source, js]
----
tinymce.init({
  selector: 'textarea',
  plugins: 'toc',
  toolbar: 'toc',
  toc_class: 'our-toc'
});
----

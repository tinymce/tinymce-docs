[[typeahead_urls]]
== `typeahead_urls`

This option allows disable the type ahead url field feature.

*Type:* `Boolean`

*Default Value:* `true`

*Possible Values:* `true`, `false`

=== Example: Using `typeahead_urls`

[source, js]
----
tinymce.init({
  selector: 'textarea',
  typeahead_urls: false
});
----

When using premium plugins without bundling, configure the `+external_plugins+` option to point to the premium plugin files. You can use different URL formats depending on your deployment:

*Relative to web-server root:*

[source,js]
----
tinymce.init({
  selector: 'textarea',
  external_plugins: {
    'advcode': '/node_modules/tinymce-premium/plugins/advcode/plugin.min.js',
    'tinycomments': '/node_modules/tinymce-premium/plugins/tinycomments/plugin.min.js'
    'licensekeymanager': '/node_modules/tinymce-premium/plugins/licensekeymanager/plugin.min.js',
  },
  plugins: 'advcode tinycomments',
  // ... other configuration
});
----

*Absolute URLs (CDN or separate server):*

[source,js]
----
tinymce.init({
  selector: 'textarea',
  external_plugins: {
    'advcode': 'https://cdn.example.com/tinymce-premium/plugins/advcode/plugin.min.js',
    'tinycomments': 'https://cdn.example.com/tinymce-premium/plugins/tinycomments/plugin.min.js'
    'licensekeymanager': 'https://cdn.example.com/tinymce-premium/plugins/licensekeymanager/plugin.min.js',
  },
  plugins: 'advcode tinycomments',
  // ... other configuration
});
----

*Relative to TinyMCE `+base_url+`:*

[source,js]
----
tinymce.init({
  selector: 'textarea',
  base_url: '/tinymce',
  external_plugins: {
    'advcode': '../../node_modules/tinymce-premium/plugins/advcode/plugin.min.js',
    'tinycomments': '../../node_modules/tinymce-premium/plugins/tinycomments/plugin.min.js'
    'licensekeymanager': '../../node_modules/tinymce-premium/plugins/licensekeymanager/plugin.min.js',
  },
  plugins: 'advcode tinycomments',
  // ... other configuration
});
----

[NOTE]
====
* Choose the URL format that matches your deployment structure.
* In production, you may need to copy plugin files to a public directory or use a CDN.
* Ensure the premium plugin files are accessible at the specified URLs.
====

For more information, see: xref:editor-important-options.adoc#external_plugins[`+external_plugins+` configuration option].

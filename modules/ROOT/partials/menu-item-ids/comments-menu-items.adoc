|===
| Menu item identifier | xref:configure/editor-appearance.adoc#examplethetinymcedefaultmenuitems[Default Menu Location] | Description

| `addcomment`
| Insert
| Adds a comment at the cursor location or for the selected text.

| `deleteallconversations`
| File
| Deletes all comments from the editor.

| `showcomments`
| View
| Shows/hides comments present in the editor.
|===

== Overview

This section shows the files required for each {productname} {editorcomponent}. The file paths shown are relative to the root {productname} package directory, where `+tinymce.min.js+` is stored. For example:

....
./
├── icons/
├── langs/
├── license.txt
├── plugins/
├── readme.txt
├── skins/
├── themes/
├── tinymce.d.ts
├── tinymce.js
├── tinymce.min.js
└── version.txt
....

[cols="1,1,4",options="header"]
|===
|Name   |Arguments    |Description

|getModel |N/A |Return a xref:#trackeddocument[TrackedDocument] `+Object+` of the document. 
|resetModel |N/A |Reset the document to its original state. 
|hasChanges |N/A |Return a boolean value indicating whether the document contains changes to be reviewed.  
|===

.Examples
[source,js]
----
// Get the current model of the document
tinymce.activeEditor.plugins.trackchanges.getModel();

// Reset the document model to its original state 
tinymce.activeEditor.plugins.trackchanges.resetModel();

// Check if document contains changes to be reviewed 
tinymce.activeEditor.plugins.trackchanges.hasChanges();
----
